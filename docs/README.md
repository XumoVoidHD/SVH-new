# Trading Bot Documentation

This folder explains how the trading bot works in simple language, so that someone without coding experience can still understand what it is doing and why.

We recommend reading the files in this order:

1. `SYSTEM_FLOW.md` – Big-picture overview of how the system runs during the day.
2. `ENTRY_RULES.md` – When and why new positions are opened.
3. `RISK_AND_SIZING.md` – How much capital is put into each trade and each sector.
4. `PROFIT_AND_STOPS.md` – How profit is taken and how losses are limited.
5. `HEDGING.md` – How the system uses inverse ETFs (like SPXU / SQQQ) to hedge the whole portfolio.
